Sobre Mí

Desarrollador Java Spring Boot Backend, con experiencia en el desarrollo de microservicios y APIs REST, utilizando herramientas como JUnit, Mockito, Jacoco y demás librerías para garantizar la calidad del código. Manejo tecnologías asociadas al control de versiones como Git y el manejo de repositorios remotos por medio de GitHub y BitBucket. He participado en el diseño de bases de datos y la creación de prototipos web, integrando soluciones tecnológicas con las necesidades del negocio. Utilizo metodologías ágiles como Scrum para la gestión de proyectos. Además, he colaborado en migraciones de sistemas utilizando herramientas como Data X, Control-M y Java APX para asegurar la estabilidad y el rendimiento del proyecto, incluyendo el desarrollo de transacciones APX y Job Batch, así como la modificación de componentes ASO.
